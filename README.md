# Project is divided into 3 main parts.
- Delivery Executive App
- Customer App
- Admin Panel

## Delivery Executive App (Flutter Project)
- Flutter Based Modern UI for login page.
- Flutter Based Modern UI for Main page.

## Customer App (Flutter Project)
- Flutter Based Modern UI for login page.
- Flutter Based Modern UI for Main page.
